# ToolJet

## Description 

- Open-source low-code framework for building React-based web applications and internal tools

## Setup 

- Dockerized setup : [Tutorial](https://docs.tooljet.com/docs/deployment/docker)
- GitHub Repo : [link](https://github.com/ToolJet/ToolJet)

## Commands to start 

- Build docker : **`docker-compose up -d`**
- Seed DB : **`docker-compose run server npm run db:seed`**
- Credentials: ID: **`dev@tooljet.io`** pass: **`password`**
